## "Run It Locally"

### 1. Clone the repo

    git clone https://github.com/akerS4I/Productivity-Dashboard.git

### 2. Install the dependencies

    cd Productivity-Dashboard
    pnpm install
    
### 3. Run on the dev server

    pnpm dev
